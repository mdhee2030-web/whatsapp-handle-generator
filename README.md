# WhatsApp Handle Generator

واجهة ويب عربية خفيفة لتوليد اقتراحات أسماء ثلاثية وتنظيم نتائج التحقق اليدوي.

> لا يتصل المشروع بواتساب، ولا يفحص توافر الأسماء أو الحسابات آلياً. هو أداة اقتراحات وحفظ محلي فقط.

## التشغيل
افتح `index.html` في أي متصفح.

## النشر عبر GitHub Pages
من صفحة المستودع:
1. Settings
2. Pages
3. اختر `Deploy from a branch`
4. اختر `main` ثم `/(root)`
5. Save

بعدها سيظهر رابط موقعك في صفحة GitHub Pages.
